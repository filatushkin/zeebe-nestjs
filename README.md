
[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.
[Zeebe](https://docs.camunda.io/docs/components/zeebe/zeebe-overview/) process automation engine powering Camunda Platform

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
